Engine Control Module -J623-, Removing and Installing

Removing

  Note

If the engine control module is replaced, perform the Guided Functions "01 - Control module, replacing" using the Vehicle Diagnostic Tester.

- Open the retainers in the direction of -arrows- and remove the Engine Control Module -J623--2-.

- Disconnect the connectors -1-.

Installing

Install in reverse order of removal and note the following:

- If the Engine Control Module -J623- is replaced, perform the "01 - mechanical engine, functions" listed adaptation using the Vehicle Diagnostic Tester.
