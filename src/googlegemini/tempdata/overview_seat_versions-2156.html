Audi A4: Overview - Seat Versions

Audi A4 (B9) 2016-2023 Service Manual / Body / Body Interior / Seat Frames / Overview - Seat Versions

Component Location Overview - Seat Versions

Overview - Front Seat

I - Standard Seat

  • Equipment:
  • Manual seat length/height and angle adjustment
  • Manual backrest angle adjuster
  • Height-adjustable headrest
  • Cargo net
  • Footwell lamp
  • Optional equipment:
  • Power seat length/height and angle adjustment
  • Power backrest angle adjuster
  • Power lumbar support adjuster
  • Front passenger side child seat anchor
  • Memory driver seat
  • Seat heating
  • RSE preparation
  • Height and depth-adjustable headrest
  • Fire extinguisher bracket
  • Overview. Refer to → Chapter "Overview - Front Seat".

II - Sport Seat

  • Equipment:
  • Manual seat forward/back, height, angle and depth adjustment
  • Manual backrest angle adjuster
  • Height adjustable headrest
  • Cargo net
  • Footwell lamp
  • Optional equipment:
  • Power seat length/height and angle adjustment
  • Power backrest angle adjuster
  • Power lumbar support adjuster
  • Front passenger side child seat anchor
  • Memory driver seat
  • Seat Heating
  • RSE preparation
  • Height and depth-adjustable headrest
  • Fire extinguisher bracket
  • Overview. Refer to → Chapter "Overview - Front Seat".

III - Sport Seat with Climate Control

  • Same equipment as the Sport seat
  • Same optional equipment as the Sport seat, including:
  • Seat ventilation with heating
  • Refer to → Chapter "Overview - Front Seat"

Overview - Rear Seats

I - Fixed Rear Bench Seat

  • Equipment:
  • Child seat anchor
  • Optional equipment:
  • Seat heating
  • Overview. Refer to → Chapter "Overview - Bench Seat/Single Seat" and → Chapter "Overview - Rear Seat Backrest, Fixed".

II - Rear Seat with Folding Backrests

  • Equipment:
  • Outer and center folding backrests
  • Child seat anchor on every seat
  • Center armrest
  • Optional equipment:
  • Storage compartment and cupholder
  • Thorax airbags
  • Seat heating
  • Overview. Refer to → Chapter "Overview - Bench Seat/Single Seat" and → Chapter "Overview - Rear Seat Backrest, Folding Backrest".

Front Seat, Mounting on Fixture for Seat Repair

Special tools and workshop equipment required

  • Engine and Transmission Holder -VAS6095A-
  • Engine/Transmission Holder - Seat Repair Fixture -VAS6136-

Procedure

  • Seat removed from the vehicle.

- Mount the -VAS6136- onto the -VAS6095A-.

- Secure the seat on the -VAS6136-.
