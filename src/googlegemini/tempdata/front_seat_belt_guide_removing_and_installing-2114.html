Audi A4: Front Seat Belt Guide, Removing and Installing

Audi A4 (B9) 2016-2023 Service Manual / Body / Body Interior / Passenger Protection, Airbags, Seat Belts / Front Seat Belt Guide, Removing and Installing

Removing

- Remove the B-pillar lower trim panel and bracket for the upper B-pillar trim panel. Refer to → Chapter "B-Pillar Lower Trim Panel, Removing and Installing".

- Push the bracket together in the direction of -arrows- and remove the belt guide -1- upward.

- Guide out the seat belt -2- and remove the seat belt guide.

Installing

Install in reverse order of removal.

Installation instructions: For example tightening specifications, replacing components. Refer to → Chapter "Overview - Front Three-Point Seat Belt".

Front Belt End Fitting, Removing and Installing

  Caution

This procedure contains mandatory replaceable parts. Refer to component overview and parts catalog prior to starting procedure.

Mandatory Replacement Parts

  • Bolt - Belt End Fitting to Body (Replace bolt if it was removed because of an accident when the seat belt was buckled)

Removing

- Move the front seat all the way forward into its highest position.

- Unclip the cover -1- on the belt relay -arrows- and fold it up.

  Caution

Risk of damaging the spring -2- if it is removed incorrectly.

  • So as not to bend the spring, only move it so far until the tab on the spring can be released.
  • Replace the three-point seat belt if the spring gets bent.

- Release the spring catches -2- by inserting a small screwdriver in the opening in the spring in the direction of -arrow-.

- Slide the belt end fitting -3- downward and disengage at the pin -1-.

Installing

Install in reverse order of removal.

Installation instructions: For example tightening specifications, replacing components. Refer to → Chapter "Overview - Front Three-Point Seat Belt".

Rear Outer Belt End Fitting, Removing and Installing

  Caution

This procedure contains mandatory replaceable parts. Refer to component overview and parts catalog prior to starting procedure.

Mandatory Replacement Parts

  • Bolt - Belt End Fitting to Floor (Replace bolt if it was removed because of an accident when the seat belt was buckled)

Removing

- Remove the rear bench seat. Refer to → Chapter "Bench Seat/Single Seat, Removing and Installing".

- Remove the bolt -2- from the belt end fitting -1-.

Installing

Install in reverse order of removal and note the following:

  • The threaded pin -arrow- on the floor panel must be positioned in the cut-out of the belt anchor.

Installation instructions: For example tightening specifications, replacing components. Refer to → Chapter "Overview - Outer Rear Three-Point Seat Belt".
