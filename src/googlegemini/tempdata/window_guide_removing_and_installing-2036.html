Audi A4: Window Guide, Removing and Installing

Audi A4 (B9) 2016-2023 Service Manual / Body / Body Exterior / Front Doors, Central Locking System / Window Guide, Removing and Installing

Removing

- Remove the door window. Refer to → Chapter "Front Door Window, Removing and Installing".

- Remove the B-pillar trim. Refer to → Chapter "B-Pillar Door Trim, Removing and Installing, Front".

- Remove the A-pillar trim. Refer to → Chapter "A-Pillar Trim, Removing and Installing".

- Remove the window guide -1- outward from the door frame -arrows-.

Installing

Install in reverse order of removal and note the following:

- Install the window guide -A- on the door flange in the sequence -1 through 5-.

  1. Position the upper edge of the window guide on the B-pillar door flange and push it forward in the direction of travel.
    • The guide pin -B- must engage in the window guide mount.
  2. Install the window guide on the door flange.
  3. Align the window guide on the A-pillar guide rail.
  4. Insert the seal -C- into the guide rail.
  5. Install the B-pillar trim and insert the seal -D- into the guide rail.

Inner Window Shaft Strip, Removing and Installing

Removing

- Remove the front window frame trim panel. Refer to → Body Interior; Rep. Gr.70; Front Door Trim Panels.

- Remove the window shaft strip -1- upward in the direction of -arrows-.

Installing

Install in reverse order of removal and note the following:

- Position the window shaft strip -1- on the window guide edge -2- at both ends -arrow- and press to the center of the door fold.

Outer Window Shaft Strip, Removing and Installing

Special tools and workshop equipment required

  • Wedge Set -T10383-
  • Wedge 1 -T10383/1-

Removing

- Move the door window into the "open" position.

- Remove the A-pillar trim. Refer to → Chapter "A-Pillar Trim, Removing and Installing".

- Using the -T10383/1-, carefully remove the window shaft strip -1- at the B-pillar trim -2- upward from the door flange -arrows-.

- Remove the window shaft strip.

Installing

TIP

The lower edge of the window shaft strip can be coated with tire mounting paste for installation.

- Align the window shaft strip -1- flush to the door body -2--arrow B- and attach it on the door flange all the way to the A-pillar in the direction of -arrows A-.

Further installation is the reverse order of removal.
